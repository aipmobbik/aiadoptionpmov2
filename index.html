AI Guideline & Prompt Playbook

A practical guide to using AI in project management to keep your projects running smoother.

AI Guideline

🚫 Do NOT Enter

  • Client names, project codes, or contract details
  • Personal data (names, IDs, salaries, performance records)
  • Financial figures specific to client engagements
  • Proprietary IP, internal processes, or trade secrets
  • Credentials, API keys, or any system access details
  • Internal audit findings or unpublished strategy

✅ Safe to Enter

  • Anonymised scenarios ("Project X", "Client A")
  • General frameworks, templates, and methodologies
  • Meeting notes after removing names & client specifics
  • Your own draft text for editing or improving
  • Published standards and PM frameworks (PMI, PRINCE2)

📤 Before Sharing AI Output

  • Verify all facts, numbers, and names are accurate
  • Remove any hallucinated data or AI-style phrasing
  • Add your professional context and judgement before sharing

Prompt Tips

Prompt Tips
Mindset for getting better output from AI every time.
01
Be Specific

The more context you give (role, goal, format), the better the output. Recommend to use the CREATE formula (C Character · R Request · E Examples · A Adjustments · T Type of Output · E Evaluation) to write prompts.

02
Specify Your Audience

Tell AI who will read the output. Executive-level vs. team-level tone changes everything.

03
AI = First Draft, You = Final Editor

Always review before using. AI can be confidently wrong — your judgement and expertise are the final filter.

04
Don't Give Up

If output isn't right, add constraints, change tone, or give a counter-example.

05
One Task at a Time

For the best results, ask one task per prompt. Split complex requests into multiple prompts.

Tag Guide

How to read the classification tags on each sample prompt in the Prompt Library.

Prompt Readiness
Generic No project-specific input required. Copy and send as-is, or with minor wording changes. Best for routine tasks where output format matters more than project context.
Specific Must be customised before use — replace all [bracket] placeholders with your actual project name, role, and documents. The more context you add, the more accurate the output.
Prompt Weight
Light Works with text input only — no file attachments needed. Type or paste key information directly into the prompt. Faster to run; suited for tasks where context fits in a few sentences.
Heavy Requires attached project documents (e.g. charter, RAID log, meeting notes). AI reads the files to generate output — quality depends on what you attach. Ensure documents are current before use.
